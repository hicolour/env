## Env! 

Quick env setup on arch linux:

* Xmonad 
* Dzen2 
* Dmenu 
* Dunst  


#### Pre-requirements

* Archlinux base base-devel instalaltion 

* Working xorg:
	

    startx





#### Setup

Clone project:

	git clone https://github.com/hicolour/env.git


Base setup: 

    make base
